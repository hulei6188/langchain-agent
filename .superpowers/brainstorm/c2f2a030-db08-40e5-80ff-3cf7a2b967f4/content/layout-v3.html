<span style="background-color: rgb(255, 255, 255)"><code>message</code></span>对象代表一条消息内容。

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| role | String | 发送该消息的实体。取值：
* **user**: 代表该消息内容是用户发送的。
* **assistant**: 代表该消息内容是 Bot 发送的。

| content | String | 消息内容。 |
| content_type | String | 消息内容的类型，取值：
* **text**: 文本类型。

当 type = **answer** 时，消息内容格式为 Markdown，即 Bot 的最终回复内容格式是 Markdown，不是纯文本。
